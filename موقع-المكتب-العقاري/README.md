# مكتب أضواء الفيصلية للخدمات العقارية

موقع عقاري احترافي متكامل باللغة العربية مع نظام إدارة (لوحة تحكم) للمشرف.

---

## 🚀 طريقة التشغيل

1. افتح ملف `index.html` في المتصفح لعرض الموقع العام
2. افتح ملف `admin.html` للوصول إلى لوحة التحكم

### بيانات الدخول الافتراضية للوحة التحكم:
```
اسم المستخدم: admin
كلمة المرور:  admin123
```

> **ملاحظة أمنية:** يُنصح بشدة بتغيير بيانات الدخول في ملف `js/data.js` قبل الاستخدام في بيئة الإنتاج.

## 🔐 طرق الوصول إلى لوحة التحكم

### 1. عبر الرابط المباشر
افتح في المتصفح: `admin.html`

### 2. عبر اختصار لوحة المفاتيح
من داخل الموقع العام، اضغط: `Ctrl + Alt + A`

### 3. عبر رابط الفوتر
رابط خفي بلون شفاف في أسفل الصفحة

---

## 🎯 ميزات لوحة التحكم

### 1. الإحصائيات
- إجمالي العقارات
- العقارات المعتمدة
- العقارات المعلقة
- طلبات العملاء
- عقارات للبيع / للإيجار
- مخططات بيانية (حسب النوع، حسب المدينة)

### 2. إدارة العقارات
- عرض جميع العقارات في جدول
- البحث في العقارات
- فلترة حسب الحالة (معتمد/معلق)
- **تعديل** أي عقار
- **حذف** أي عقار
- **عرض** العقار في الموقع
- **موافقة** على العقارات المعلقة

### 3. العقارات المعلقة
- عرض العقارات بانتظار الموافقة
- عرض تفاصيل كاملة (مالك، هاتف، وصف)
- موافقة بنقرة واحدة
- تعديل قبل الموافقة
- رفض (حذف)

### 4. طلبات العملاء
- عرض جميع طلبات "اطلب عقارك"
- تفاصيل العميل (اسم، هاتف، بريد)
- التواصل المباشر (اتصال، واتساب، بريد)
- تغيير الحالة: جديد → تم التواصل → مغلق
- حذف الطلبات

### 5. إضافة عقار جديد
- نموذج كامل بنفس حقول الموقع
- العقار يُنشر مباشرة (معتمد)
- خيار "عقار مميز" للظهور في الواجهة

---

## 🔄 نظام العمل

### للعقارات المضافة من الموقع (الزوار):
1. الزائر يملأ نموذج "أضف عقارك"
2. العقار يُحفظ بحالة **"معلق"** (لا يظهر في الموقع)
3. رسالة تأكيد: "سيتم مراجعة العقار من قبل الإدارة"
4. المشرف يراجع العقار في لوحة التحكم
5. المشرف يوافق → العقار يظهر في الموقع
6. أو المشرف يحذف/يرفض العقار

### للعقارات المضافة من لوحة التحكم (المشرف):
1. المشرف يضيف العقار من "إضافة عقار"
2. العقار يُنشر **مباشرة** بحالة "معتمد"
3. يظهر في الموقع فوراً

## 💾 تخزين البيانات

جميع البيانات تُخزن في `localStorage` المتصفح:
- `properties` - جميع العقارات
- `propertyRequests` - طلبات العملاء
- `favorites` - العقارات المفضلة للزوار
- `adminSession` - جلسة المشرف (صالحة 24 ساعة)

### لتصدير البيانات:
- البيانات محفوظة في المتصفح فقط
- لمشاركة البيانات بين الأجهزة، يلزم نقل `localStorage`
- للإنتاج الفعلي، يُفضل استخدام backend (مثل PHP/Node.js) مع قاعدة بيانات

## 🎨 التخصيص

### تغيير بيانات الدخول:
افتح `js/data.js` وعدّل:
```js
const ADMIN_CREDENTIALS = {
    username: 'admin',     // غيّر اسم المستخدم
    password: 'admin123'   // غيّر كلمة المرور
};
```

### تغيير ألوان الموقع:
افتح `css/style.css` وغير المتغيرات:
```css
:root {
    --navy-900: #0a1f3d;    /* الكحلي الداكن */
    --gold-500: #c9a96e;    /* الذهبي */
    --beige-100: #faf7f0;   /* البيج الفاتح */
}
```

### إضافة مدن جديدة:
1. افتح `js/data.js` وأضف المدينة في `CITIES`
2. افتح `index.html` وأضفها في `<select>` و `chips`
3. افتح `admin.html` وأضفها في نماذج الإضافة والتعديل
